# trjones1.github.io

Personal site and interactive CV for **Tramel Jones** — Engineering Leader, Full Stack Builder, AI Creator.

Live at [trjones1.github.io](https://trjones1.github.io)

## Sections

- **Hero** — Animated role rotation, stats, and quick links
- **Tech Stack** — Infinite marquee + categorized skills
- **Experience** — Interactive career timeline
- **Projects** — ChatVerse, LexiBot, Daily Hustle Coach
- **GitHub** — Live profile and repo feed via GitHub API
- **About** — Leadership philosophy and education
- **Beyond Engineering** — Personal interests

## Resume

A printable resume is available at [`/resume/`](resume/index.html). Use the browser's print dialog to save as PDF.

To use a custom PDF, add `resume/Tramel-Jones-Resume.pdf` and update the download links in `index.html`.

## Local development

This is a static site — no build step required. Serve locally with any static file server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Customize

- Update content in `index.html`
- Styles in `css/main.css`
- Interactivity in `js/main.js`
- Profile photo: `img/profile.png`
- LinkedIn URL: search for `linkedin.com/in/trjones1` in `index.html`
